Block & Inline Elements in HTML

1. Block Element.

In HTML A block element (or block-level element) is an element that occupies the full width available and begins on a new line. This means that a block element will stack vertically in the layout of a web page.

e.g- <div>,<h1>

Inline Element

In HTML, inline Elements are a type of element that do not Start on a new line and only take up as much width as necessary. They typically only contain text or other initial elements, and they allow other element to let basic them on the same line.
